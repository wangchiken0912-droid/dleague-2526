# D LEAGUE｜台南夢達七人足球聯賽 25/26 官方網站（初版）

這是使用 Next.js + TypeScript + Tailwind CSS 建立的基本專案骨架，
目前已經包含：

- 首頁：顯示聯賽基本資訊與「下一場比賽」
- `data/league.config.ts`：聯賽設定
- `data/teams.ts`：球隊資料
- `data/fixtures.ts`：2025-10-19 首輪賽程

## 本機啟動方式

1. 安裝套件：

```bash
npm install
```

2. 啟動開發伺服器：

```bash
npm run dev
```

3. 瀏覽器開啟：<http://localhost:3000>

之後可以逐步擴充：

- 完整賽季賽程
- 積分榜頁面
- 球隊介紹頁
- 新聞與影片模組
